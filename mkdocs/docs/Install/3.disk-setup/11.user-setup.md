# User Setup

## Adding a User

```bash
# Create a new user
useradd -m -G users,wheel,audio -s /bin/bash ric
# Set the password for the user
passwd ric
# Add the user ric to the group video
gpasswd -a ric video
```

## Allowing sudo

Uncomment the following line in /etc/sudoers
```bash
%wheel ALL=(ALL:ALL) ALL
```

## View System Logs

To view system logs
```bash
journalctl
```

To get system load
```bash
btop
```
